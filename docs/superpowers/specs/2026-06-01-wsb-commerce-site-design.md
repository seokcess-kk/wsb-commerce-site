## 1. 개요 & 목표

- **무엇:** WSB(우리스마트바이오)가 운영하는 건강기능식품 D2C **종합몰**. 신규 브랜드 **NUTROGIN 3종(FOCUS / CLEAR / REST)** + 기존 건기식 몇 종 판매.
- **왜:** 오픈마켓 의존 탈피, 자사 회원·매출 데이터 직접 확보, 브랜드 경험 전달.
- **브랜드 구조:** WSB = 종합몰의 운영 주체(그릇), NUTROGIN = 입점 대표 브랜드.
- **기조:** ~4주 내 오픈(6/30). "보고 → 담고 → 결제 → 받고 → 재방문" 핵심 매출 흐름에 집중.

## 2. 범위

**v1 포함**
- 상품 카탈로그 / 상품 상세(PDP) / 효능별 내비게이션·필터
- 장바구니 / 단건 결제 / 회원·비회원 주문
- 마이페이지(주문·배송조회, 회원정보, 내 리뷰, 보유 쿠폰)
- 리뷰(포토리뷰 포함) / 1:1 문의 / FAQ·공지
- 기본 쿠폰(발급·적용)
- 브랜드 스토리(WSB · NUTROGIN)
- **운영자 어드민**: 상품관리 / 카테고리 / 주문관리 / 회원관리 / 리뷰·문의 / 쿠폰 / **콘텐츠 관리(메인 히어로 배너 캐러셀·띠배너·홈 진열·공지·FAQ)** / 대시보드
  - 메인 배너: 이미지(PC/모바일 분리)·문구·링크·노출기간·순서(드래그)·ON·OFF를 어드민에서 직접 제어
  - v1 제외: 페이지 구조를 자유 편집하는 페이지빌더(→ 2차)

**2차 보류 (명시적 제외)**
- 정기구독(정기배송), 회원등급제, 진단형 추천, B2B/도매, 알림톡·CRM 자동화, 옴니채널/오프라인 연동, 라이브커머스, 다국어·다통화

## 3. 사용자 & 핵심 플로우

- **구매자:** 홈/검색 → 상품목록(효능 필터) → 상품상세 → 장바구니 → 주문/결제 → 주문완료 → 마이페이지 재방문/리뷰
- **운영자:** 어드민 로그인 → 상품 등록·재고 관리 → 주문 처리(상태변경·송장) → CS 응대 → 대시보드 모니터링

## 4. 정보구조(IA)

- **고객몰:** 홈 · 상품(효능별: 두뇌·집중 / 면역 / 수면 / 활력 …) · 상품상세 · 브랜드(WSB·NUTROGIN) · 장바구니/주문 · 마이페이지 · 고객지원 · 로그인/가입
- **메인 내비 기준:** 효능·기능별 (NUTROGIN은 '두뇌·집중'의 간판)
- **어드민:** 대시보드 · 상품관리 · 카테고리 · 주문관리 · 회원관리 · 리뷰/문의 · 쿠폰 · 배너/공지 · 설정
- **푸터:** 사업자정보 · 통신판매업 신고번호 · 개인정보처리방침 · 이용약관 · 건강기능식품 심의/고지 문구

## 5. 데이터 모델 스케치

- `Product` (효능 카테고리, 성분·원산지·인증 정보, 심의 문구 포함)

- `Category` (효능 기반)
- `User` (회원) / 비회원 주문 지원
- `Order` · `OrderItem` · `Payment` · `Shipment`(송장)
- `Review`(포토리뷰) · `Inquiry`(1:1 문의)
- `Coupon` · `UserCoupon`
- `Banner` · `Notice`
- `AdminUser` (권한)

## 6. 기술 아키텍처

- **프레임워크:** Next.js (App Router, Server Components 중심) + **Vercel** 배포
- **데이터/백엔드:** **Supabase** — Postgres DB + **Supabase Auth**(소셜 로그인 포함) + **Supabase Storage**(상품·상세 이미지) + RLS 보안
  - ORM: Drizzle 또는 Supabase 클라이언트 직접 사용 (구현 단계 확정)
- **UI:** Tailwind CSS + shadcn/ui, 디자인 토큰으로 브랜드 컬러 주입
- **결제:** 토스페이먼츠 SDK (결제 레이어는 ON/OFF 토글 가능하게 설계 — §0 컨틴전시)
- **분석:** GA4 + (선택) Microsoft Clarity

## 7. 결제·배송·회원 정책

- **결제수단:** 신용카드 · 간편결제(토스/카카오/네이버페이) · 계좌이체/가상계좌
- **배송:** 택배 연동, 기본배송비 3,000원 / 5만원 이상 무료, 도서산간 추가. 송장은 수기·엑셀 입력으로 시작(API 연동은 2차)
- **반품/교환:** 수령 후 7일. 건기식 개봉·훼손 시 제한 — 상세 고지
- **회원:** 회원 + 비회원 주문 둘 다 허용. 소셜 로그인 카카오·네이버. 가입 혜택 쿠폰

## 8. 건강기능식품 규제·법적 준수

- **신고·서류(클라이언트 책임 / 제작 측 지원):** 건강기능식품판매업 신고, 통신판매업 신고, 구매안전(에스크로) 확인증
- **표시·광고:** 기능성 표시·광고 자율심의(건협) 통과 문구만 사용. 상세페이지에 **심의필 표시 + "질병의 예방·치료를 위한 것이 아님" 고지** 영역 표준 배치
- **PG 심사:** 건강기능식품 카테고리 등록, 상품 5종 이상
- **개인정보:** 처리방침·이용약관 비치, 14세 미만 정책(성인 대상이나 동의 영역 마련), 쿠키 안내
